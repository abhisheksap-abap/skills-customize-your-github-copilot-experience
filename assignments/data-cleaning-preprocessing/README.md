# 📘 Assignment: Data Cleaning & Preprocessing

## 🎯 Objective

Learn to clean and preprocess real-world datasets in Python. Students will handle missing values, detect and treat outliers, and transform data for further analysis or modeling.

## 📝 Tasks

### 🛠️ Handle Missing Values

#### Description
Identify missing values in a dataset and apply appropriate strategies to deal with them (imputation, removal, or flagging).

#### Requirements
Completed program should:
- Detect missing values in columns
- Apply at least two methods to handle missing data
- Summarize the impact of each method

### 🛠️ Detect and Treat Outliers

#### Description
Find outliers in the dataset and decide how to handle them (remove, cap, or transform).

#### Requirements
Completed program should:
- Identify outliers using statistical methods (e.g., IQR, z-score)
- Apply at least one method to treat outliers
- Visualize before and after treatment

### 🛠️ Data Transformation

#### Description
Transform features for better analysis, including normalization, scaling, or encoding categorical variables.

#### Requirements
Completed program should:
- Normalize or scale numerical features
- Encode categorical variables
- Document the transformation steps
